继电脑软件后，安卓软件自热也有。

最早接触的是手机，那时候能给手机刷双系统的我，却还不会给电脑重载系统，原因很简单，我没有电脑；
后来发现，电脑装机比手机容易，也安全多了；
再后来，刷机渐渐少了，玩电脑的时间反倒越来越长了，
那些曾以为是“黑科技”的软件，也就越来越少收集了。

我有些收集瘾，但还不至于我用不到的软件也去收集测试，所以我手机软件的分类不多，原可以分在影音播放、下载工具、文件管理的软件，我通通放在一个“实用”类，因为这些软件，基本上就是一类只留下一两个。很多软件旧版跟新版没啥不一样，一般也就不特别更新了。

如果需要一个mod版软件仓库搜寻想要的软件，推荐到这个仓库寻找

实用修改版软件

Accu_Battery

Accu_Battery v1.1.8c

早期一直开着，用来跟踪电池健康度，现在主要用来偶尔监测电流，测试快充线跟快充头。

ADM Pro下载器

ADM Pro下载器_v5.1.2(251)
用了六年的多线程下载器，当时下载刷机包实在是太爽了。
直到发现他下不了m3u8的视频，才换成了IDM+。

IDM+

IDM+
多线程下载工具。
除了名字，与电脑的IDM没有半毛钱关系，但是真的好用。

Root Explorer

Root Explorer-v4.5.2

大名鼎鼎RE管理器，我root后必装的文件管理器，主要用于修改system分区的文件。
虽然现在有着诸如ES、NP、MT等功能更多的文件管理器，但是强大的同时带来的是繁杂，习惯了RE管理器的简洁与操作模式，再也不愿意换而已。

ES文件管理器

ES File Explorer-Premium-v4.2.2.4

之前在局域网传输方案中有提到ES文件管理器，现在主要用来清理垃圾与实现局域网传输。

Nova Launcher

Nova Launcher_v6.2.13

最早是因为能自定义图标而看上他，我的桌面并没有弄得花里胡哨，只是他能自定义行列数、图标大小等细节参数，很舒服。

附上两个我很喜欢的图标包

miracles

iflat

SD Maid

SD Maid v4.11.9 耗子修改★高级解锁版

存储空间清理工具，绝对比系统自带的清理工具强大。

钛备份

TitaniumBackup-v8.4.0.2

著名备份软件，主要是能备份如微信聊天记录等应用数据，换手机时就很好用。
在微信取消了云端备份聊天记录后，除了备份到电脑上，就只能靠钛备份了。
需要root。

WPS

WPS Office_v12.6.0解锁高级版

不得不说手机上WPS要比微软的Office舒服，关键是轻量。
国际版，破解是破解了，但需要登录后才能使用pdf转word等高级功能。

Zarchiver

Zarchiver_v0.9.4

压缩解压工具。
虽然RE管理器或一些手机自带就支持文件的压缩/解压缩，但是还得有个支持格式足够齐全的专业压缩软件以备不时之需。

静读天下

本地阅读器，虽然有几个在线书库，但基本是国外的，不适合用来追还在连载的小说。
在，我忘了是哪个版本，连kindle特有的azw3格式也能阅读后，几乎就是全能型阅读器。
唯一的缺点就是设置非常繁琐，第一次上手有一定难度，自定义程度极高是他的优点也是他的缺点，但是设置好以后，阅读体验极佳。
如果嫌繁琐，多看阅读是不错的选择。

6.0以后目录从居中式变成了侧边式。

静读天下v6_0_0

静读天下_v5_2_8

wifi万能钥匙

不得不说到了一些地方，急用的时候还挺好用。
万能钥匙_4.3.10.a

讯飞语记

搭配静读天下的朗读功能。

讯飞语记_4.4.1281_kill

谷歌相机

本来想专门写一篇来介绍Google Camera，但是看了下别人的文章，自认为写不出更好的，具体原理可以看知乎上的回答Google Camera的 HDR+算法有多厉害？

简单地说，白天可能看不出什么差别，但是到了晚上，谷歌相机就是神。

自然要放上Google Camera Port的连接

trCamera_7.2.010_V7_汉化版

相机修图

修图软件不独立出来，是真的不好找。

VSCO

早期我最喜欢的胶片滤镜，种类极其多，甚至支持HSL调整，效果也十分优秀，现在懒了，懒得从百多个滤镜中调一个出来了，就用FIMO为主。

Lightroom-v5.2.2-Mod-arm64

Lightroom-v5.2.2-Mod-arm64

早期主力修图工具是snapseed，现在是LR。
稍微高级点的修图自然离不开调整曲线与HSL，泼辣修图等也可以做到，但是习惯了电脑的PS后，上手手机端的LR就很简单。
手机端的PS功能不怎么多，但是LR却足够强大。

MIX
